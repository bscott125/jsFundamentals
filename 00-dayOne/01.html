NOTES

  • When using Emmet, class is notated by a "." and id is by "#"
  • HTML stands for Hypertext Markup Langauage
  • DOCTYPE states the type of document. In this case, HTML.
  • our html tag denotes our language to be English, per the lang="en"
  • The head tag houses metadata, or detailed data about our doc.
  • UTF-8 specifies character encoding. Each character(letter, number, even spaces) has a numeric value.
  • Viewport notes the device we are reading our document in.
  • title reflects the name that displays on our browser tab.
  • link allows us to connect to other files, such as our stylesheet (CSS)
  • a style tag can help us inject css into our document.
  • the body tag houses all the other tags that we want to display in our browser. This will be what the user sees!
